PROYECTO DE ENTORNOS DE DESARROLLO PT1

DESCRIPCIÓN

Este proyecto se trata de una simulación de una aplicación para la gestión de una tienda de informática en la que se tendrá que gestionar la información de cada cliente, producto (ya sea físico o digital) y pedido que se realice.
Todo esto se llevará a cabo mediante el uso de la programación orientada a objeto (POO), que permitirá una mejor estructuración y organización y del código.

REQUISITOS E INSTRUCCIONES DE EJECUCIÓN

Requisitos:
Se requiere de VS code junto con las extensiones Plant UML y UML Generator for Java.

Instrucciones de ejecución:
Se debe clonar el repositorio y ejecutar el main, el cual como salida esperada debería tener un resumen de un pedido de dos artículos de un cliente.
